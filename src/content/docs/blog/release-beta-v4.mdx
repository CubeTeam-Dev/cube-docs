# Cube v4.0 ベータ版 Webダッシュボード公開！

こんにちは！
v3.0に続く新たなアップデートとして、**Webダッシュボード（v4.0 ベータ版）** を公開しました！

## 新機能：Webダッシュボード

これまでDiscord上で行っていた設定の一部を、ブラウザから**視覚的・直感的**に管理できるようになりました。
今回のアップデートは、今後の大型機能追加に向けた重要な基盤となります。

- **ダッシュボード：** [https://cubeteam.app](https://cubeteam.app)
- **現在利用できる機能：** Discord連携ログイン / サーバー選択 / 一般設定（Botの言語設定など）
- **使い方・ガイド：** [Cube Docs](https://docs.cubeteam.app/)

<Aside type="caution">
  本リリースはベータ版です。現時点では設定可能な項目が限られています。
  予期せぬバグや表示崩れを見つけた場合は、公式Discordの不具合報告チャンネルまでご連絡ください。
</Aside>

## 次回以降のアップデート予告（v4.1〜）

今回のダッシュボード公開は、これから始まる進化の第一歩です。
次回の **v4.1** では、ダッシュボード上で強力な **Anti-Raid（荒らし対策）** や **AutoMod** のカスタマイズが可能になる予定です。
さらにその先には、革新的なAIモデレーション機能も控えています。ぜひご期待ください！
