import { Aside, Tabs, TabItem } from "@astrojs/starlight/components";

Discord のメッセージを**右クリック**（スマートフォンの場合は長押し）→ **[アプリ]** から実行できる機能です。コマンドを入力せずに、その場でメッセージに対して操作できます。

---

<Tabs>

<TabItem label="翻訳 (Translate)">

メッセージを瞬時にサーバー設定の言語に翻訳します。翻訳結果は自分だけに表示されます（エフェメラルメッセージ）。

**使い方**

1. 翻訳したいメッセージを**右クリック**（長押し）
2. **[アプリ]** をタップ
3. **[Translate]** を選択

**特徴**

- テキスト・埋め込みメッセージ（Embed）・ボタンのラベルまで含めて翻訳します
- 翻訳結果には「元のメッセージへ移動」ボタンが付属します
- サーバー外（DM・グループDM）でも使用できます

</TabItem>

<TabItem label="これより先を削除 (Purge below)">

選択したメッセージ自身と、それ以降に送信されたメッセージを一括削除します。

**使い方**

1. 削除の起点にするメッセージを**右クリック**（長押し）
2. **[アプリ]** をタップ
3. **[Purge-below]** を選択
4. 確認ボタンをクリック

**削除モードの選択**

- **はい** — 起点のメッセージ以降、最大100件を全て削除します
- **カスタム** — 削除する件数を数値で指定できます

<Aside type="caution">

この機能を使用するには **メッセージの管理** 権限が必要です。

</Aside>

</TabItem>

</Tabs>
