---
title: VC 移動

---

import { Aside } from "@astrojs/starlight/components";

ボイスチャンネル（VC）に接続中のメンバーを、別のボイスチャンネルへ移動させるコマンドです。

<Aside type="caution">

すべての `/move` コマンドには **メンバーを移動する** 権限が必要です。

</Aside>

---

### /move one — 特定のメンバーを移動する

指定した1人のメンバーを別のボイスチャンネルに移動します。

| オプション | 必須 | 説明 |
|---|---|---|
| `member` | ✅ | 移動するメンバー |
| `channel` | ✅ | 移動先のボイスチャンネル |

**実行例**
```
/move one member:@たろう channel:#雑談VC
```

<Aside type="note">

対象メンバーがボイスチャンネルに接続していない場合は移動できません。

</Aside>

---

### /move all — 自分と同じ VC の全員を移動する

コマンドを実行した人が接続しているボイスチャンネルの**全員**を、指定したチャンネルに移動します。

| オプション | 必須 | 説明 |
|---|---|---|
| `channel` | ✅ | 移動先のボイスチャンネル |

**実行例**
```
/move all channel:#メインVC
```

<Aside type="note">

このコマンドを使うには、実行者自身がボイスチャンネルに接続している必要があります。

</Aside>

---

### /move multiple — 指定した VC の全員を別の VC に移動する

指定したボイスチャンネルに接続している全員を、別のボイスチャンネルに一括移動します。

| オプション | 必須 | 説明 |
|---|---|---|
| `from_channel` | ✅ | 移動元のボイスチャンネル |
| `to_channel` | ✅ | 移動先のボイスチャンネル |

**実行例**
```
/move multiple from_channel:#旧VC to_channel:#新VC
```

`/move all` と違い、実行者がボイスチャンネルに参加していなくても操作できます。
