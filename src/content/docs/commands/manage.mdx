Cube Bot はサーバー運営を効率化するための強力な管理ツールを提供しています。

<Aside type="note">
  管理コマンドの多くは、実行者に特定の権限が必要です。
  権限不足の場合はエラーが表示されますので、適切な権限を持っているかご確認ください。
</Aside>

## メンバー操作

### move <Badge text="メンバーを移動" variant="caution" />
メンバーを別のボイスチャンネルに移動します。

| オプション | 説明 |
| :--- | :--- |
| `member` | 移動させる対象のメンバー |
| `channel` | 移動先となるボイスチャンネル |

### ban <Badge text="メンバーをBAN" variant="danger" />
ユーザーをサーバーから BAN (追放) します。

| オプション | 説明 |
| :--- | :--- |
| `user` | BAN する対象のユーザー |
| `reason` | BAN の具体的な理由 |

---

### kick <Badge text="メンバーをキック" variant="danger" />
ユーザーをサーバーからキック (一時追放) します。

| オプション | 説明 |
| :--- | :--- |
| `user` | キックする対象のユーザー |
| `reason` | キックの具体的な理由 |

---

### timeout <Badge text="メンバーをタイムアウト" variant="caution" />
ユーザーを一定時間話せなくします。

| オプション | 説明 |
| :--- | :--- |
| `user` | タイムアウトする対象のユーザー |
| `duration` | タイムアウト時間（例: 1s, 5m, 1h, 3d） |
| `reason` | タイムアウトの具体的な理由 |

---

## ロール管理

### role <Badge text="ロールの管理" variant="caution" />
ロールの付与・削除・確認を一括で行います。

| サブコマンド | 説明 |
| :--- | :--- |
| `give` | ユーザーに特定のロールを与えます。 |
| `remove` | ユーザーから特定のロールを剥奪します。 |
| `list` | サーバー内の全ロールのリストを表示します。 |
| `members` | 指定したロールを持つ全メンバーを表示します。 |
| `multiple` | 条件を指定してロールを一括付与・削除します。 |

---

## 自動化機能

### auto_purge <Badge text="メッセージの管理" variant="caution" />
指定した間隔でチャンネル内のメッセージを自動削除します。

| オプション | 説明 |
| :--- | :--- |
| `channel` | 自動削除を有効にするチャンネル。 |
| `limit` | 一回の削除実行で削除する最大メッセージ数。 |
| `interval` | 削除を実行する間隔（例: 5s, 1m, 1h）。 |

---

### auto_react <Badge text="管理者" variant="danger" />
特定のチャンネルへのメッセージ投稿に対して、自動的にリアクションを追加します。

| オプション | 説明 |
| :--- | :--- |
| `channel` | 自動リアクションを有効にするチャンネル。 |
| `emoji` | 追加する絵文字。 |
