### /ping — Bot の状態を確認する

Cube Bot が正常に動作しているか確認できます。ネットワークのレイテンシや稼働時間などの情報が表示されます。

**実行例**
```
/ping
```

以下の情報が表示されます。

| 項目 | 説明 |
|---|---|
| ネットワーク遅延 | WebSocket と REST API のレイテンシ（ms） |
| シャード情報 | 接続中のシャード ID とその状態 |
| Bot 情報 | 稼働時間、参加サーバー数、シャード数 |
| 応答時間 | コマンドの総処理時間（ms） |

レイテンシは以下の基準で評価されます。

| 状態 | レイテンシ |
|---|---|
| 🟢 Excellent | 100ms 未満 |
| 🟡 Good | 100〜200ms |
| 🔴 Poor | 200ms 以上 |
