Cube Bot の基本的なコマンドです。Bot の動作状況やステータスを確認できます。

## ping

Bot の遅延（レイテンシ）と現在のステータスを表示します。

| 項目 | 説明 |
| :--- | :--- |
| **ネットワーク遅延** | WebSocket と REST API の遅延。ステータス判定も表示されます。 |
| **シャード情報** | サーバーが接続しているシャードの ID と WebSocket 遅延。 |
| **Bot情報** | Bot の合計サーバー数、合計シャード数、および起動時間。 |
| **応答時間** | コマンドを受け取ってから表示するまでの合計応答時間（往復）。 |

### ネットワーク遅延の詳細
<Aside type="note">
  ネットワーク遅延は、Discord サーバーと Bot サーバー間の通信速度を示します。
  通常、100ms 前後であれば快適に動作します。
</Aside>

### 起動時間の表示
Bot が最後にメンテナンス等で再起動してからの経過時間を表示します。
長期稼働の安定性を確認するために役立ちます。
