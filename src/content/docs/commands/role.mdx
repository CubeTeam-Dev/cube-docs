## 個別のロール操作

### /role give — ロールを付与する

指定したメンバーにロールを付与します。

| オプション | 必須 | 説明 |
|---|---|---|
| `member` | ✅ | ロールを付与するメンバー |
| `role` | ✅ | 付与するロール |

**実行例**
```
/role give member:@たろう role:@VIPメンバー
```

<Aside type="caution">

必要な権限: **ロールを管理する**

</Aside>

---

### /role remove — ロールを削除する

指定したメンバーからロールを取り外します。

| オプション | 必須 | 説明 |
|---|---|---|
| `member` | ✅ | ロールを削除するメンバー |
| `role` | ✅ | 削除するロール |

**実行例**
```
/role remove member:@たろう role:@VIPメンバー
```

<Aside type="caution">

必要な権限: **ロールを管理する**

</Aside>

## ロールの情報を確認する

### /role list — ロール一覧を表示する

サーバーに存在するロールの一覧をページ形式で表示します。

| オプション | 必須 | 説明 |
|---|---|---|
| `page` | — | 表示するページ番号（デフォルト: 1） |

**実行例**
```
/role list
/role list page:2
```

---

### /role members — ロールの保有メンバーを表示する

指定したロールを持つメンバーの一覧を表示します。

| オプション | 必須 | 説明 |
|---|---|---|
| `role` | ✅ | 確認するロール |
| `page` | — | 表示するページ番号（デフォルト: 1） |

**実行例**
```
/role members role:@VIPメンバー
```

---

## 一括でロール操作する

### /role multiple — ロールを一括付与・削除する

サーバー内の複数のメンバーに対して、ロールをまとめて付与または削除します。

| オプション | 必須 | 説明 |
|---|---|---|
| `role` | ✅ | 付与または削除するロール |
| `type` | ✅ | 操作の種類（`give` または `remove`） |
| `target` | ✅ | 対象（下記参照） |
| `required_role` | — | `target: require` 使用時に必要なロール |

**`target` オプションの選択肢**

| 値 | 対象 |
|---|---|
| `all` | サーバー全員 |
| `human` | Botを除く全ユーザー |
| `bot` | Botのみ |
| `require` | `required_role` で指定したロールを持つメンバーのみ |

**実行例**
```
/role multiple role:@メンバー type:give target:human
/role multiple role:@イベント参加者 type:give target:require required_role:@参加登録済み
```

<Aside type="note">

対象メンバーが多い場合、処理に時間がかかることがあります。また、Bot 自身のロールより上位のロールは操作できません。

</Aside>

<Aside type="caution">

必要な権限: **ロールを管理する**

</Aside>
