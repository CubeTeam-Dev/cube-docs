Cube Bot の動作をサーバーに合わせてカスタマイズするための設定コマンドです。

## ロール管理 (高度設定)

### role_keeper
メンバーがサーバーを抜けて再参加した際に、以前持っていたロールを自動的に復元する機能です。

1. ロールキーパーの設定を行うチャンネルでコマンドを入力します。
2. `enabled` オプションでロールキーパーを有効にします。
3. `manage_permissions` や `moderation_permissions` で、特定のロールを復元対象から除外するか設定します。

<Aside type="note">
  ロールキーパー機能によって復元されるのは、Bot がそのサーバーに参加していた間のロール履歴に基づきます。
</Aside>

## 翻訳機能

### auto_translate
特定のチャンネルでの発言を自動的に指定された言語に翻訳します。

| 言語コード | 対象言語 |
| :--- | :--- |
| `JA` | 日本語 |
| `EN` | 英語 |
| `FR` | フランス語 |
| `DE` | ドイツ語 |
| `ES` | スペイン語 |

<Aside type="caution">
  自動翻訳を有効にすると、そのチャンネルの全メッセージが翻訳され、メッセージの下に翻訳結果が表示されます。
</Aside>

## 言語設定

### language
サーバー内での Cube Bot の言語設定を変更します。

| 言語コード | 対象言語 |
| :--- | :--- |
| `ja` | 日本語 (デフォルト) |
| `en` | 英語 |

<Aside type="note">
  サーバーのデフォルト言語によって、Bot の応答メッセージ（成功・失敗・エラーメッセージなど）が切り替わります。
</Aside>
