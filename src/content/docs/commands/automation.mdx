チャンネルの日常的な管理を自動化するコマンドです。すべてのコマンドには **チャンネルを管理する** 権限が必要です。

---

## 自動削除（Auto Purge）

チャンネルのメッセージを、指定した件数と間隔で定期的に自動削除します。ルールや告知チャンネルを常に最新の状態に保つのに便利です。

### /auto_purge set — 自動削除を設定する

| オプション | 必須 | 説明 |
|---|---|---|
| `limit` | ✅ | 1回の削除で消す最大メッセージ件数 |
| `interval` | ✅ | 削除を実行する間隔（例: `5s`, `2m`, `1h`） |
| `channel` | — | 設定するチャンネル（省略時は実行したチャンネル） |

**時間の書式**

- `s` — 秒（例: `30s`）
- `m` — 分（例: `5m`）
- `h` — 時間（例: `1h`）

**実行例**
```
/auto_purge set limit:100 interval:1h
/auto_purge set limit:50 interval:30m channel:#お知らせ
```

---

### /auto_purge remove — 自動削除を解除する

設定した自動削除を解除します。

| オプション | 必須 | 説明 |
|---|---|---|
| `channel` | ✅ | 解除するチャンネル |

**実行例**
```
/auto_purge remove channel:#お知らせ
```

## 自動リアクション（Auto React）

チャンネルに投稿されたメッセージに、指定した絵文字を自動でリアクションします。投票チャンネルや盛り上げに活用できます。

### /auto_react set — 自動リアクションを設定する

| オプション | 必須 | 説明 |
|---|---|---|
| `emoji` | ✅ | リアクションする絵文字（例: `👍`, カスタム絵文字も使用可） |
| `channel` | — | 設定するチャンネル（省略時は実行したチャンネル） |

**実行例**
```
/auto_react set emoji:👍
/auto_react set emoji:👍 channel:#投票チャンネル
```

<Aside type="note">

カスタム絵文字を使用する場合は `<:絵文字名:絵文字ID>` の形式で入力してください。

</Aside>

---

### /auto_react remove — 自動リアクションを解除する

設定した自動リアクションを解除します。

| オプション | 必須 | 説明 |
|---|---|---|
| `emoji` | ✅ | 解除する絵文字 |
| `channel` | — | 解除するチャンネル（省略時は実行したチャンネル） |

**実行例**
```
/auto_react remove emoji:👍
```

## スティッキーメッセージ（Sticky）

チャンネルの下部に常にメッセージを表示し続ける機能です。ルールや注意事項をチャンネルに固定しておくのに便利です。

### /sticky set — スティッキーを設定する

チャンネルの一番下に常に表示されるメッセージを設定します。新しいメッセージが来るたびに Bot がメッセージを再送信して常に最下部に表示されます。

| オプション | 必須 | 説明 |
|---|---|---|
| `content` | ✅ | 固定するメッセージの内容 |
| `attachment` | — | 一緒に固定する画像ファイル |
| `channel` | — | 設定するチャンネル（省略時は実行したチャンネル） |

**実行例**
```
/sticky set content:このチャンネルでは日本語のみ使用してください。
/sticky set content:今週のイベント案内 attachment:event.png channel:#お知らせ
```

---

### /sticky delete — スティッキーを削除する

設定したスティッキーメッセージを削除します。

| オプション | 必須 | 説明 |
|---|---|---|
| `channel` | — | 削除するチャンネル（省略時は実行したチャンネル） |

**実行例**
```
/sticky delete
/sticky delete channel:#お知らせ
```
