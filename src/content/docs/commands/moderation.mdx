サーバーのルール違反への対応やチャンネルの緊急制限に使うコマンドです。すべてのコマンドは、対象メンバーより**上位のロール**を持つユーザーのみ実行できます。

---

## メンバーへの対応

### /setnick — メンバーのニックネームを変更する

サーバーのメンバーのニックネームを変更します。

| オプション | 必須 | 説明                           |
| ---------- | ---- | ------------------------------ |
| `user`     | ✅   | ニックネームを変更するメンバー |
| `nickname` | ✅   | 新しいニックネーム             |
| `reason`   | ✅   | ニックネームを変更した理由     |

**実行例**

```
/setnick user:@対象メンバー nickname:新しいニックネーム
```

<Aside type="caution">

必要な権限: **ニックネームを変更する**

</Aside>

---

### /ban — メンバーを BAN する

サーバーからメンバーを永久に追放し、再参加を禁止します。

| オプション | 必須 | 説明                             |
| ---------- | ---- | -------------------------------- |
| `user`     | ✅   | BAN するメンバー                 |
| `reason`   | —    | BAN の理由（監査ログに残ります） |

**実行例**

```
/ban user:@スパムユーザー reason:スパム行為のため
```

<Aside type="caution">

必要な権限: **メンバーをBANする**

</Aside>

---

### /unban — BAN を解除する

BAN されたユーザーのBANを解除し、再参加を許可します。

| オプション | 必須 | 説明                   |
| ---------- | ---- | ---------------------- |
| `user`     | ✅   | BAN を解除するユーザー |

**実行例**

```
/unban user:@解除するユーザー
```

<Aside type="caution">

必要な権限: **メンバーをBANする**

</Aside>

---

### /kick — メンバーをキックする

サーバーからメンバーを退場させます。キックされたメンバーは、招待リンクなどから再参加できます。

| オプション | 必須 | 説明                               |
| ---------- | ---- | ---------------------------------- |
| `user`     | ✅   | キックするメンバー                 |
| `reason`   | —    | キックの理由（監査ログに残ります） |

**実行例**

```
/kick user:@問題のあるメンバー reason:注意喚起無視のため
```

<Aside type="caution">

必要な権限: **メンバーをキックする**

</Aside>

---

### /timeout — メンバーをタイムアウトする

指定した時間、メンバーがメッセージを送信できない状態にします。最長 **28日間** 設定できます。

| オプション | 必須 | 説明                                        |
| ---------- | ---- | ------------------------------------------- |
| `user`     | ✅   | タイムアウトするメンバー                    |
| `duration` | ✅   | タイムアウトの時間（例: `10m`, `2h`, `1d`） |
| `reason`   | —    | 理由                                        |

**時間の書式**

- `s` — 秒（例: `30s`）
- `m` — 分（例: `10m`）
- `h` — 時間（例: `2h`）
- `d` — 日（例: `1d`）

**実行例**

```
/timeout user:@荒らしメンバー duration:1h reason:荒らし行為
```

<Aside type="caution">

必要な権限: **メンバーをタイムアウト**

</Aside>

---

### /untimeout — タイムアウトを解除する

タイムアウト中のメンバーの制限を手動で解除します。

| オプション | 必須 | 説明                           |
| ---------- | ---- | ------------------------------ |
| `user`     | ✅   | タイムアウトを解除するメンバー |

**実行例**

```
/untimeout user:@タイムアウト中のメンバー
```

<Aside type="caution">

必要な権限: **メンバーをタイムアウト**

</Aside>

## チャンネルの制御

### /lock — チャンネルをロックする

チャンネルの全員（`@everyone`）に「メッセージを送信」権限を拒否し、新しいメッセージを送れなくします。

| オプション | 必須 | 説明                                               |
| ---------- | ---- | -------------------------------------------------- |
| `channel`  | —    | ロックするチャンネル（省略時は実行したチャンネル） |

**実行例**

```
/lock
/lock channel:#緊急チャンネル
```

<Aside type="caution">

必要な権限: **チャンネルを管理する**

</Aside>

<Aside type="note">

ロックを解除するには `/unlock` コマンドを使用してください。

</Aside>

---

### /unlock — チャンネルのロックを解除する

`/lock` でロックしたチャンネルのロックを解除します。

| オプション | 必須 | 説明                                                   |
| ---------- | ---- | ------------------------------------------------------ |
| `channel`  | —    | ロック解除するチャンネル（省略時は実行したチャンネル） |

**実行例**

```
/unlock
/unlock channel:#緊急チャンネル
```

<Aside type="caution">

必要な権限: **チャンネルを管理する**

</Aside>

---

### /slowmode — スローモードを設定する

チャンネルにスローモードを設定し、メンバーが連続してメッセージを送れる間隔を制限します。最長 **6時間** まで設定できます。

| オプション | 必須 | 説明                                       |
| ---------- | ---- | ------------------------------------------ |
| `time`     | ✅   | スローモードの間隔（例: `5s`, `2m`, `1h`） |

**実行例**

```
/slowmode time:30s
/slowmode time:2m
```

<Aside type="note">

スローモードを解除するには `/slowmode time:0s` を実行してください。

</Aside>

<Aside type="caution">

必要な権限: **チャンネルを管理する**

</Aside>
