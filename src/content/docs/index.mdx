## Cube でできること

<CardGrid>
  <Card title="モデレーション" icon="approve-check-circle">
    BAN・キック・タイムアウトからチャンネルのロックまで、サーバーの秩序を守るコマンドを完備しています。

    [詳細を見る →](/commands/moderation)

  </Card>
  <Card title="ロール管理" icon="puzzle">
    ロールの付与・削除はもちろん、サーバー全員への一括操作も可能。再参加時にロールを自動復元するロールキーパー機能も搭載。

    [詳細を見る →](/commands/role)

  </Card>
  <Card title="自動化機能" icon="rocket">
    自動削除・自動リアクション・スティッキーメッセージで、チャンネルの管理を省力化できます。

    [詳細を見る →](/commands/automation)

  </Card>
  <Card title="翻訳・グローバルチャット" icon="translate">
    メッセージの翻訳や、複数のサーバーをつなぐグローバルチャット機能で、言語の壁を超えたコミュニティ運営を支援します。

    [詳細を見る →](/commands/settings)

  </Card>
  <Card title="AI モデレーション" icon="star">
    AI を活用したスパム検知・有害コンテンツのフィルタリング。段階的なワークフローによる高精度な自動モデレーション。

    *(近日公開予定)*

  </Card>
</CardGrid>
